## Script Issues

### Scripts fail to run
- Ensure required dependencies are installed (`pip install -r requirements.txt` where applicable)
- Check Python version compatibility (Python 3.8+ recommended)
- Verify file permissions: `chmod +x *.sh` for shell scripts

### AI analysis produces errors
- Verify AI API keys are configured correctly
- Check network connectivity for external API calls
- Review script logs for specific error messages
