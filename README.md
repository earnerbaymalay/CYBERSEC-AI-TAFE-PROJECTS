# TAFE Cyber Security & AI Portfolio
### Project Location: Perth, Western Australia (2026)

This repository contains a suite of automated security tools developed in a mobile-first environment (iSH/Alpine Linux on iOS). 

---

## 🛠 Project 1: Sentinel-AI (Threat Assessment)
**Sentinel-AI** is a Python-based log analyzer using the **Gemini 3 Flash** API to identify unauthorized access patterns.

## 📡 Project 2: Mobile-Recon (Network Discovery)
**Mobile-Recon** is a Bash utility built with **Nmap**. It uses **TCP Connect Scans** (-sT) to function within iOS kernel restrictions.

## 🔐 Project 3: Crypt-Vault (AES-256 Encryption)
**Crypt-Vault** is a data-at-rest security tool that implements symmetric encryption for sensitive local files using **AES-256**.

---

## 💻 Environment & Tools
- **OS:** Alpine Linux (via iSH) / iOS 15.8.6
- **Hardware:** iPhone 7 (A10 Fusion)
- **Languages:** Python 3.x, Bash
